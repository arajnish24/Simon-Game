Game Rules

  • Press any keys to start the game.
  • A color sequence will appear.
  • Repeat the sequence by clicking the buttons.
  • With each round, the sequence gets longer.
  • If you click the wrong color, the game ends.
  • Press any key to restart after a game over.
  • If you played well then you won the game.
